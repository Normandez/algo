# algo
C++ Realization of Some Algorithms

# Algorithms
* Reverse Array
* Caesar Encrypt (Caesar Cipher)
* XOR Cipher
* Fibonacci Sequence
* Factorial Sequence
* RLE Compression
* Permutation Generation
* Catalan number (the number of possible unique BSTs or the number of possible open/close parentheses)
